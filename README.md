# NIS-2 & BSI IT-Grundschutz Audit Portfolio

Willkommen auf meinem Portfolio-Repository. Dieses Repository demonstriert meine Kompetenz in der Überprüfung der Konformität nach der **EU-Richtlinie NIS-2 (EU 2022/2555)** unter Anwendung der **BSI IT-Grundschutz-Methodik**.

Das Portfolio zeigt, wie ich als Auditor die organisatorischen und technischen Maßnahmen gemäß Artikel 21 NIS-2 prüfe, basierend auf den Schutzbedarfskategorien (Normal, Hoch, Sehr Hoch) des BSI.

## Fokus & Kompetenz

* **Schwerpunkt:** Prüfung von Governance, Risikomanagement-Maßnahmen und Meldeprozessen.
* **Methodik:** Abgleich der internen Richtlinien mit NIS-2 und BSIG.

## Repository-Struktur

| Ordner | Inhalt |
| :--- | :--- |
| `00_Governance_ISMS` | ISMS, Risikoanalyse, Audit-Planung (Hoch/Sehr Hoch) |
| `01_Techn_Massnahmen` | Patchmanagement, Pentests, Endpoint Security (Normal/Hoch) |
| `02_Incident_BCM` | Incident Response, BCM, Backup Management (Normal/Hoch) |
| `03_Lieferkette_Personal`| Supply Chain, Awareness, Zugriffskontrolle (Normal/Hoch) |
